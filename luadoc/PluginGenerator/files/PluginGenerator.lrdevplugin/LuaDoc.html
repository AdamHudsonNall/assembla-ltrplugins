LuaDoc:replace (new, start, stop)
Replace lines in new from start to stop (legacy function comment) with luadoc compatible comment.

Parameters:

  • new: array of lines containing entire lua file under construction.
  • start: line number (new index) of top line of legacy function comment.
  • stop: line number (new index) of bottom line of legacy function comment.

Usage:

    Presently just finds first non empty line and prefixes it with '---', then puts '--' 
    in front of all subsequent lines until it gets to the function def.
